# Kenya Pending Bills & SME Cashflow Transparency Starter

A lightweight data transparency and analysis toolkit for tracking pending government and institutional bills affecting SME liquidity.

## Why this exists (Feb 2026)
Delayed payments to SMEs create cashflow strain, increase borrowing costs, and slow economic growth. This repository provides structured datasets, reconciliation workflows, and simple analysis templates.
